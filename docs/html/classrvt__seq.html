Detailed Description

template<typename REQ = rvt_item, typename RSP = REQ>
class rvt_seq< REQ, RSP >

Base sequence class. All sequences must be extended from this base class. Does not have a virtual sequencer set.

Template Parameters
REQRequest item type. Defaults to <rvt_item>
RSPResponse item type. Defaults to REQ
